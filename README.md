# ba4d
A custom set of programs to play Bad Apple!! (ft Nomico) PV on Microsoft DOS (tested on MS-DOS 6.22 and FreeDOS 1.3)

## Installation
Copy the `copyme` folder to your DOS disk (or USB stick, it will be easier that way if you have CSM and can use USB sticks for DOS). The thing is about 45MiB, so you will need an at least 200MB disk if you have no other programs.

Rename `copyme` to something like `ba4d`.

## Running
```COMMAND.COM
cd ba4d\
REM or whatever your folder is called
.\MASTER
```
You need a decent PC to be able to run it, ie anything from lets say at least 1998.
